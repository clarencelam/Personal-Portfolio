Clam Game

A web game based on running a food stand on Maple Island. A fun project with the purpose of learning to code in vanilla Javascript. Built from scratch.

LIVE DEMO SOURCE CODE
Tech Stack
  • Built with Javascript, HTML, and CSS
  • Version control and deployment with Git
Technical Achievments
  • Created gameloop with a stabilized frame rate of 100fps on all computers/browsers. Did this by implementing a gameloop which ingests the current timestamp, and checks the difference between the last timestamp. If the change in time is >= 10 milliseconds, the next frame is updated. As a new frame is loaded 100 times a second, the game framerate is cemented at 100 frames per second regardless of the user's browser speed.

  • Implemented a simple gamestate design pattern to manage the different phases of the game. At any given time, one gamestate is active (like a Pause gamestate, Menu gamestate, etc.), and going to other gamestates changes elements like the background image, active functions, etc.
